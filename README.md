# 📚 LibRova — Online Book Borrowing Platform

A seamless and modern web application that digitizes the traditional library experience. Users can explore a vast collection of books, filter by categories, and borrow titles digitally.

---

## 🌐 Live URL

 **httpfs://lib-rova.vercel.app/**

---

## 🎯 Purpose

LibRova brings the library experience online. Users can browse 30+ curated books across 8 categories, borrow instantly after logging in, search by title, and manage their reading profile — all in a beautifully crafted modern-classic aesthetic.

---

## ✨ Key Features

- 🔐 **Secure Authentication** — Email/password login & Google OAuth via BetterAuth
- 📚 **38 Book Catalog** — Story, Tech, Science, Self Help, Fantasy, History, Biography categories
- 🔍 **Search Functionality** — Real-time title search across all books
- 🗂️ **Category Sidebar** — Filter books by category with URL-based navigation
- 📖 **Book Detail Pages** — Private routes, only accessible to logged-in users
- 👤 **User Profile** — View and update profile information
- ✏️ **Profile Update** — Name & photo URL update using BetterAuth
- ⭐ **Reader Reviews** — Testimonials section with real book references
- 📊 **Stats Section** — Platform statistics showcase
- 🎠 **New Arrivals** — Featured books section on homepage
- 📱 **Fully Responsive** — Mobile, tablet, and desktop layouts
- 🔔 **Toast Notifications** — HeroUI toast for all user feedback
